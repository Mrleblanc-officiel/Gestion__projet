# Mise en place du repos Git
    1- Création du repertoire via GitHub
    
    2- Liasons du Depot via la commande Git clone <url https>

    3- Git status pour voir les modifications de validation et permet de voir les fichier suivis ou non suivie

    4- Git add . : permet d'inclure ce qui sera valider prend en compte tous les modification fait (ou a la place du <.> nous pourrions spécifier un fichier spécifique)

    5- Git commit -m "commentaire ou description " : toujours spécifier une description a chaques modifications faits

    6- Git push : heberge les commits locaux vers Github
